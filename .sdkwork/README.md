# SDKWork CMS Workspace Metadata

This directory stores source-controlled SDKWork workspace metadata for `sdkwork-cms`.

Allowed content:
- Repository-local skills and plugins.
- Non-secret workspace manifests.
- Documentation that helps agents and operators use the project.

Forbidden content:
- Runtime databases, logs, caches, secrets, tokens, private keys, generated SDK transport output, and user-private files.

Related specs:
- `../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md`
- `../sdkwork-specs/SOUL.md`
