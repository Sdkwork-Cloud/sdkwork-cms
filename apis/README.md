# API Contracts

This directory contains authored CMS API contracts and API review inputs.

Allowed content:
- OpenAPI documents.
- Route manifests.
- Schemas, examples, changelogs, and validation fixtures.

Forbidden content:
- Generated SDK transport output.
- Runtime controller, handler, service, or repository implementation.
