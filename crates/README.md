# Rust Crates

This directory contains CMS Rust backend crates:

- `sdkwork-content-cms-service`: business use cases, domain contracts, and service ports.
- `sdkwork-content-cms-repository-sqlx`: SQLx repository implementation and database schema constants.
- `sdkwork-routes-cms-open-api`: open-api route adapter.
- `sdkwork-routes-cms-app-api`: app-api route adapter.
- `sdkwork-routes-cms-backend-api`: backend-api route adapter.
