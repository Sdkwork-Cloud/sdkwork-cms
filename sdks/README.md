# SDK Families

This directory contains CMS SDK family metadata and generation inputs.

SDK families:
- `sdkwork-cms-sdk`: open-api/public integration SDK.
- `sdkwork-cms-app-sdk`: app-api/user-facing delivery SDK.
- `sdkwork-cms-backend-sdk`: backend-api/admin and operator SDK.

Generated transport output must stay under each family `generated/server-openapi` directory after running the canonical SDKWork generator.
